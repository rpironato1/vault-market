# UI Spec — Landing (`/`)

## Objetivo
Apresentar o Vault Market e direcionar usuários para o catálogo de NFTs.

## Público-alvo
Não autenticado (público).

## Componentes principais
- Hero (headline + value proposition)
- Seção “Como funciona” (3 passos)
- Seção “Por que confiar”
- CTA “Explorar NFTs”
- FAQ curto e links legais

## Estados
- Normal
- Offline (mensagem simples e CTA para retry)

## Ações
- CTA primário: “Explorar NFTs”
- CTA secundário: “Entrar / Criar conta”

## Copy constraints
- Não usar linguagem de depósito/aposta
- Explicar: compra NFT → VaultCoins → experiências → prêmios USDT → saque

## Analytics
- `ui_landing_viewed`
- `ui_landing_cta_market_clicked`
