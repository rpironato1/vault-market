# UI Spec — Gift Cards (`/app/gift-cards`)

## Objetivo
Oferecer itens/benefícios complementares (se mantido no produto).

## Componentes
- Lista de gift cards
- Detalhe com regras
- CTA para comprar (se aplicável) ou resgatar

## Regras
- Precisa estar alinhado ao PRD (evitar “lucro garantido” sem amarrações legais/risco).
- Se permanecer, tratar como item utilitário com termos claros.

## Dados necessários
- `GET /gift-cards`
- `POST /gift-cards/purchase` (se aplicável)
